# Dotfiles for i3/polybar/rofi

### Dependencies:

- i3
- rofi
- polybar
- maim
- xclip
- vim
- xterm

## Installation:

  ```bash
  git clone https://github.com/Azuremuzzet/mini-dotfiles.git
  cd mini-dotfiles
  ./fast-cp
  ```
