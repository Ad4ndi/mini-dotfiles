# Dotfiles for i3/polybar/rofi
Greened fork of https://github.com/Ad4ndi/mini-dotfiles

### Dependencies:

- i3
- rofi
- polybar
- maim
- xclip
- vim
- xterm

## Installation:

  ```bash
  git clone https://github.com/Azuremuzzet/mini-dotfiles.git
  cd mini-dotfiles
  ./fast-cp
  ```
